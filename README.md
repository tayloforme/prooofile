# YouTube Shorts Downloader

Простой локальный сайт: вставляешь до 4 ссылок на YouTube Shorts — браузер скачивает каждое видео как `.mp4`.

## Требования

- Node.js 18+
- [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) в PATH

Установка yt-dlp:

```bash
pip install yt-dlp
# или
brew install yt-dlp
```

## Запуск

```bash
npm install
npm start
```

Открой http://localhost:8000
